# `@curong/process`

`@curong/process` 主要包含以下的方法:

- `exec`: `Promise` 版本的 `exec` 方法
- `open`: 使用用户操作系统定义的默认应用程序打开文件
- `sleep`: 睡眠一段时间
- `spawn`: `Promise` 版本的 `spawn` 方法
